# 💰 Expense Tracker

A modern and responsive expense tracker built using **React**, **Zustand**, and **Tailwind CSS**.  
Easily manage your expenses with filtering, sorting, and a clean UI.

---

## 🚀 Features

✅ Add, edit and delete expenses  
✅ Search and filter by category  
✅ Sort expenses (date, amount, name)  
✅ Light/Dark theme toggle 🌙☀️  
✅ Responsive design (mobile + desktop)  
✅ Clean UI with smooth interactions

---

## 🛠 Tech Stack

- React (Vite)
- Zustand (state management)
- Tailwind CSS (styling)
- Lucide Icons

---

## 📦 Installation

```bash
git clone https://github.com/your-username/expense-tracker-app.git
cd expense-tracker-app
npm install
npm run dev
```
